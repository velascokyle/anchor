Install in 60 seconds
Closest thing to a “web store” install without the web store.
⌄
  1. Download anchor-extension-v1.0.0.zip from the button above.
  2. Unzip it. You should see a folder that contains manifest.json.
  3. Open Chrome → chrome://extensions
  4. Turn on Developer mode (top-right).
  5. Click Load unpacked → select the folder that has manifest.json.
  6. Pin Anchor from the puzzle icon so it’s always visible.

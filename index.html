What Anchor does

Anchor isn’t a “motivation” tool. It’s a friction layer that shows up at the exact moment discipline usually fails — when you’re tilted.

Detects emotional patterns

Consecutive losses, max daily loss, and bursts of overtrading. You choose the mode.

Forces a cooldown

Overlay blocks impulsive continuation and pushes you into a short reset protocol.

Stays private

Runs locally. No accounts. No cloud dashboards. TradingView data processed on-device.

How it works

Simple loop: read realized P&L changes, update session state, trigger cooldown overlay, resume when cleared.

1

Read realized P&L

Anchor listens for P&L updates and trade closures so it stays aligned with your actual realized results.

2

Update session state

Consecutive losses + daily loss limit update instantly in the background service worker.

3

Trigger cooldown

If your configured threshold hits, Anchor throws the overlay and starts the cooldown timer.

4

Resume clean

After the timer and/or journal prompt, state returns to Active and you continue with clarity.

Install

This is the fastest install possible without the Chrome Web Store (Web Store requires a review/publisher setup).

Install in Chrome ✓
  1. Click Download.
  2. Unzip the file (you should get a folder like anchor-extension).
  3. Open chrome://extensions
  4. Turn on Developer mode (top-right).
  5. Click Load unpacked → select the unzipped folder (the one that contains manifest.json).
Quick “is it working?” test +
  1. Go to TradingView in a tab.
  2. Open the Anchor extension popup.
  3. Click Debug: Trigger Test (overlay should appear on the TradingView tab).
Troubleshooting +
  • If Chrome says it can’t find icons, confirm your icons/ folder is inside the loaded folder and paths match the manifest.
  • If “Load unpacked” fails, make sure you selected the folder that contains manifest.json (not the parent folder).

FAQ

Short answers, no fluff.

Why not the Chrome Web Store yet? +
Web Store publishing requires a developer account + review process. This page is the fastest “ship it” path.
Does it track unrealized P&L? +
Anchor uses realized outcomes (trade closure / realized P&L changes) because that’s the cleanest source of truth.
Is my data sent anywhere? +
No. The extension processes TradingView data locally in your browser.

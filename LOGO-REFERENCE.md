# ⚓ Anchor Logo - Visual Reference

## Icon Sizes

The Anchor logo is available in multiple sizes for different use cases:

### Extension Icons (PNG)
- **16×16px** - Chrome toolbar icon (what you see in the browser)
- **48×48px** - Extension management page
- **128×128px** - Chrome Web Store listing
- **512×512px** - High-resolution promotional use

### Logo Design

**Background:** Dark navy gradient (#141a24)
**Anchor:** Light blue-gray (#8ca0c8)
**Style:** Clean, minimal maritime anchor

**Components:**
```
    ⚪  ← Ring (for chain/rope)
    |
    |   ← Vertical shaft
    |
   ─┼─  ← Horizontal crossbar
    |
   ⚓   ← Curved flukes with angled ends
```

## Where You'll See It

### 1. Chrome Toolbar
The 16×16px icon appears next to your address bar when the extension is installed.

### 2. Extension Menu
The 48×48px icon appears in chrome://extensions/ page.

### 3. Overlay (When Triggered)
The Anchor SVG logo appears on the locked state overlay with the message "Session Ended".

### 4. Chrome Web Store
The 128×128px icon appears in the store listing.

## Design Philosophy

The anchor symbol represents:
- **Stability** - Staying grounded during market volatility
- **Discipline** - Anchoring to your trading rules
- **Professional Standards** - Institutional approach for individual traders
- **Calm Mastery** - Not drifting from deliberate execution

## Color Meaning

**Dark Navy Background (#141a24)**
- Professional, serious, institutional
- Not flashy or gamified
- Calm, focused environment

**Light Blue-Gray Anchor (#8ca0c8)**
- Calm authority (not aggressive)
- Technical precision
- Trust and reliability
- Visible at all sizes

## Technical Specs

**File Format:** PNG (primary), SVG (source)
**Transparency:** No transparency (solid background for consistency)
**Corner Radius:** 15% of size (rounded but not circular)
**Stroke Width:** Scales proportionally with icon size
**Anti-aliasing:** Enabled for smooth edges

## Chrome Web Store Requirements ✅

- ✅ 128×128px icon (icon128.png)
- ✅ High-res 512×512px (icon512.png)
- ✅ Solid background (no transparency issues)
- ✅ Clear at small sizes (16px still legible)
- ✅ Consistent branding across all sizes

## Additional Assets Needed for Chrome Web Store

**Promotional Images** (create separately):
1. Small tile: 440×280px
2. Large tile: 920×680px
3. Marquee: 1400×560px

**Screenshots** (take from extension):
- At least 1, up to 5 screenshots
- Recommended: 1280×800 or 640×400
- Show: Overlay, Calendar, Journal, Settings

**Optional:**
- Video demo (YouTube link)
- Detailed description with screenshots

## Brand Consistency

**Always use:**
- Official Anchor logo (don't recreate or modify)
- Dark background version for digital
- Proper anchor color (#8ca0c8)

**Never:**
- Stretch or distort the logo
- Change the anchor color (unless grayscale)
- Add effects (shadows, glows, gradients to the anchor itself)
- Rotate or tilt
